std::optional<bytes> evmc::from_hex ( std::string_view  hex)
inline

Decodes hex encoded string to bytes.

In case the input is invalid the returned value is std::nullopt. This can happen if a non-hex digit or odd number of digits is encountered. Whitespace in the input is ignored.
